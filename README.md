# Frontwars_autoplacer
# Installation
  You may need to install pyautogui if it it not already installed
  ```
  pip install pyautogui
  ```
  Or on linux and mac
  ```
  python3 -m pip install pyautogui
  ```
  if you have any more problems please go to documentation for [pyautogui](https://pyautogui.readthedocs.io/en/latest/install.html)

# Use
  At line 20 in the file named "Frontwarsautoplacer.py"
  ```
  place(7, 100)
  ```
  put in the number of what you want it to place and how meny you want, be aware this will click your mouse for you so do not put it at a number to high, also there maybe a few clicks lost due to ping issues and delay in the game it sould be quite few however and if you are having a lrage number of clicks voided report it
