# Frontwars_autoplacer
# Installation
  You may need to install pyautogui if it it not already installed
  ```
  pip install pyautogui
  ```
  Or on linux and mac
  ```
  python3 -m pip install pyautogui
  ```
  if you have any more problems please go to documentation for [pyautogui](https://pyautogui.readthedocs.io/en/latest/install.html)

# Use
  When you run the code a dialog will send in the terminal, follow the message for formatting options
# Clarifucation
  This is not made to replace middle clicking but to allow you mostly to quickly send out bombs/ships aswell you can let this run in the backround if you leave *This is slower for placing houses, ports and factorys.
